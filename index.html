bhavyTr

A lightweight tensor library in C/C++ designed to give you full control over memory, stride, and tensor computation.

Read Docs Get Library

Why bhavyTr?

Modern frameworks hide complexity. bhavyTr exposes it. Built for developers who want to truly understand how tensors work internally — from memory layout to stride computation.

Low-Level Control

Direct access to tensor memory, shape, and stride.

Educational Design

Learn how tensor libraries actually work under the hood.

Lightweight

Minimal and fast with no unnecessary abstractions.

How to Use bhavyTr

Download the latest release, extract it, and integrate bhavyTr into your C/C++ project using compiled binaries.

📁 Step 1: Download & Extract

bhavyTr/
├── include/
├── lib/
│   ├── gcc/
│   ├── msvc/
├── bin/
│   ├── gcc/
│   ├── msvc/

⚙️ Step 2: Compile with Binary

GCC (Linux / MinGW)

gcc main.c -Iinclude -Llib/gcc -lbhavyTr -o app
./app

MSVC (Windows)

cl main.c /I include lib/msvc/bhavyTr.lib
app.exe

📌 Step 3: Example Code

#include <stdio.h>
#include "tensor.h"

int main() {
    int shape[] = {2, 3};

    Tensor t = createTensor(shape, 2);

    printTensor(&t);

    freeTensor(&t);

    return 0;
}

⚠️ Important Notes

  • Do not mix GCC and MSVC builds
  • Always include -Iinclude during compilation
  • Use compiled binaries instead of source compilation
  • Ensure correct linking with static library

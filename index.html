Why BhavyTr?

BhavyTr is a lightweight and high-performance tensor engine written in C. It is designed for modular architecture with a clean separation between user layer, parser, and core engine, making it efficient and scalable.

Features

⚡ High Performance

Built in C for maximum speed and efficiency.

🧩 Modular Design

Clean separation of parser and core engine.

🔌 Easy Integration

Simple API for C projects.

Architecture 

User Code → Parser → Core Engine → Compute Layer

Quick Example 

#include "BhavyTr.h"

int main() {
    Tensor *t = createTensor("[[1,2],[3,4]]");
    printTensor(t);
}

Beautifully Simple C Syntax

Stop manually calculating strides and allocating flat arrays. BhavyTr's integrated parser detects dimensions and shapes automatically.

  • → Zero external dependencies
  • → C23 Standard Compliance
  • → Single-header API entry 

#include "BhavyTr.h"

int main() {
    // One line. Automatic shape.
    Tensor t = createTensor("L[[1,2],[3,4]]");

    int first = GET_VAL(t, 0, 0);
    
    printf("Core Value: %d", first);

    freeTensor(&t);
    return 0;
}

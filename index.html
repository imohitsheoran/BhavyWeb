Why BhavyTr?

Built for performance. Made to last.

A clean, modular architecture that separates concerns so you can build on top of it with confidence.

⚡

High Performance

Pure C gives you direct hardware access with zero overhead. No garbage collection, no runtime surprises.

🧩

Modular Design

Clean separation between the user API, parser, and core engine. Extend any layer without touching the others.

🔌

Easy Integration

Drop the headers into your project and link the static library. Works in any existing C codebase.

📐

Tensor Slicing

NumPy-style indexing: single elements, full slices, range slices, and negative indexing — all supported.

📡

Broadcasting

Automatically broadcasts tensors of compatible shapes, just like you'd expect from a modern tensor library.

🔬

Expression Engine

Register named tensors and evaluate expressions like ops("A+B") as strings at runtime.

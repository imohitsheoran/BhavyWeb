Documentation

Learn how to install, configure, compile, and use BhavyTr in your C projects.

Installation

Download the latest release and extract it into your project directory. 

bhavytr/
├── include/
├── core/
├── parser/
└── BhavyTr.h

How to Compile Your Project

Use the following commands to compile BhavyTr depending on your system.

🐧 Linux (GCC)

gcc main.c -Iinclude -Llib -lbhavytr -o app
./app

🪟 Windows (MinGW)

gcc main.c -Iinclude -Llib -lbhavytr -o app.exe
app.exe

🪟 Windows (MSVC)

cl main.c /I include /link libbhavytr.lib
app.exe

Folder Structure

  • include/ → Public headers
  • parser/ → Input parsing layer
  • core/ → Tensor engine

Basic Usage

#include "BhavyTr.h"

int main() {
    Tensor *t = createTensor("[[1,2],[3,4]]");
    printTensor(t);
    return 0;
}

API Example

Tensor *a = createTensor("[[1,2],[3,4]]");
Tensor *b = createTensor("[[5,6],[7,8]]");

Tensor *c = addTensor(a, b);

printTensor(c);

Notes

  • Currently supports CPU execution
  • GPU support planned in future versions
  • Ensure proper memory management

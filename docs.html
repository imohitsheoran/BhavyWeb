Documentation

Learn how to install, configure, and use BhavyTr in your C projects.

Installation

Download the latest release and extract it into your project directory. 

bhavytr/
├── include/
├── core/
├── parser/
└── BhavyTr.h

Folder Structure

BhavyTr is divided into modular layers for better maintainability:

  • include/ → Public headers
  • parser/ → Input parsing layer
  • core/ → Tensor engine

Basic Usage

Include the main header and start using tensors: 

#include "BhavyTr.h"

int main() {
    Tensor *t = createTensor("[[1,2],[3,4]]");
    printTensor(t);
    return 0;
}

API Example

Example of tensor creation and operations: 

Tensor *a = createTensor("[[1,2],[3,4]]");
Tensor *b = createTensor("[[5,6],[7,8]]");

Tensor *c = addTensor(a, b);

printTensor(c);

Notes

  • Currently supports CPU execution
  • GPU support planned in future versions
  • Ensure proper memory management

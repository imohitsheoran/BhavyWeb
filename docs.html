Documentation

Learn how to install, configure, compile, and use BhavyTr in your C projects.

Installation

Download the latest release and extract it into your project directory. 

bhavytr/
├── include/
├── core/
├── parser/
└── BhavyTr.h

Setup & Compilation

Follow these steps to prepare your project and compile BhavyTr correctly.

📁 Step 1: Folder Structure

BhavyTr/
│   main.c
│   libbhavytr_shared.dll
│
├── include/
└── lib/
    └── gcc/
        libbhavytr_shared.dll.a
        libbhavytr_static.a

📋 Step 2: Copy Files (Windows)

mkdir BhavyTr

xcopy /E /I BhavyTr-v1.0.2\include BhavyTr\include

mkdir BhavyTr\lib\gcc

copy BhavyTr-v1.0.2\lib\gcc\libbhavytr_shared.dll.a BhavyTr\lib\gcc\
copy BhavyTr-v1.0.2\lib\gcc\libbhavytr_static.a BhavyTr\lib\gcc\

copy BhavyTr-v1.0.2\bin\gcc\libbhavytr_shared.dll BhavyTr\

⚙️ Step 3: Compile (Dynamic)

gcc main.c -I include -L lib/gcc -lbhavytr_shared -o app.exe

⚡ Step 4: Compile (Static - Recommended)

gcc main.c -I include lib/gcc/libbhavytr_static.a -o app.exe

🚀 Step 5: Run

app.exe

Make sure libbhavytr_shared.dll is in the same folder as your executable.

Folder Structure

  • include/ → Public headers
  • parser/ → Input parsing layer
  • core/ → Tensor engine

Basic Usage

#include "BhavyTr.h"

int main() {
    Tensor *t = createTensor("[[1,2],[3,4]]");
    printTensor(t);
    return 0;
}

API Example

Tensor *a = createTensor("[[1,2],[3,4]]");
Tensor *b = createTensor("[[5,6],[7,8]]");

Tensor *c = addTensor(a, b);

printTensor(c);

Notes

  • Currently supports CPU execution
  • GPU support planned in future versions
  • Ensure proper memory management

Tensor Creation

Tensor* createTensor(int32_t* shape, int ndim, DataType dtype)

Creates a new tensor with allocated memory initialized to zero. This is the base constructor used internally by most other APIs.

Parameters

shape (int32_t*)
Pointer to an array describing tensor dimensions.
Example: {2, 3} → 2x3 matrix

ndim (int)
Number of dimensions.
Must match length of shape array.

dtype (DataType)
Specifies element type: F32, F64, I32, I64, U8, BOOL

Returns

Pointer to allocated Tensor object.
Returns NULL if memory allocation fails.

Memory Behavior

Allocates new memory for both metadata and data buffer.
Caller is responsible for calling freeTensor().

When to Use

  • Creating empty tensors
  • Base for manual initialization
  • Internal library construction

Example


int32_t shape[] = {2, 3};

Tensor* t = createTensor(shape, 2, F32);

printTensor(t);

freeTensor(t);

Notes

  • Data is always contiguous
  • Initialized to zero
  • Invalid shape → undefined behavior

Copy / View

Tensor* copyTensor(Tensor* t)

Creates a deep copy of the tensor.

Parameters

t (Tensor*) → Source tensor

Returns

New tensor with independent memory

Why Use

Use when you need a separate tensor that does not affect the original.

Memory

Allocates new buffer → O(n)


Tensor* copy = copyTensor(original);
Tensor* viewTensor(Tensor* t, int32_t* shape, int ndim)

Creates a new tensor that shares the same data buffer. No memory copy is performed.

Parameters

t → Source tensor

shape → New shape

ndim → New dimensions

Returns

New tensor wrapper (shared memory)

Why Use

  • Fast reshape
  • Memory efficiency

Important Rule

Total elements must match.

Example


Tensor* v = viewTensor(t, (int32_t[]){1,6}, 2);

Danger

  • Modifying original affects view
  • Free original after view → invalid memory
Tensor* reshapeTensor(Tensor* t, int32_t* shape, int ndim)

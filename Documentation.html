1. Product Overview

BhavyTr v1.0.1P is a unified tensor computation library built in C. It bridges the gap between high-performance backends and intuitive frontend syntax.

The Core Concept

Unlike libraries that force you to define shapes manually, BhavyTr uses an integrated parser to detect dimensions directly from the L[...] macro syntax.

2. Architecture

// Logic Flow
User Code → createTensor(L[[...]])
Parser Layer → Detects Rank & Shape
Core Engine → Allocates Contiguous Memory

4.1 createTensor

Initialize a tensor using the L macro. The library supports 1D vectors and 2D matrices in the current version.

1D Syntax createTensor("L[1, 2, 3]");
2D Syntax createTensor("L[[1,2],[3,4]]");

4.2 freeTensor

⚠️

Manual Memory Management

Tensors are allocated on the heap. You must free them to prevent memory leaks, especially when working with loops or recursive AI logic. 

// Requires the address of the tensor object
freeTensor(&myTensor);

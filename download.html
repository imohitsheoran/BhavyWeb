Getting Started

Installation & Setup

Follow these steps to get BhavyTr running in your project.

📦

Installation

  • 01 Visit the website bhavy.org
  • 02 Download the ZIP file from the releases page
  • 03 Extract the ZIP file into your project folder
⚙️

Setup

  • → Create a new project folder
  • → Add lib and include folders from the extracted archive
  • → Create a generate output folder
  • → Windows: Add the bin/ folder containing .dll files
  • → Linux: Place files in the folder containing your C source file

Project Layout

Folder Structure

A clean layout keeps your project organised. Here's the recommended directory structure for your BhavyTr project.

lib/ Static library files
include/ Header files for BhavyTr
main.c Your application source
l.bhavyatr.dll Windows shared library
project/
my-project/
├── lib/
│ └── libBhavyTr.a
├── include/
│ └── BhavyTr.h
├── main.c
└── l.bhavyatr.dll

Build

Compile Your Project

Open a terminal and run the appropriate command for your operating system.

01
Open Terminal
Open Command Prompt or PowerShell in your project directory.
02
Run GCC Command
Compile your source file and link against BhavyTr.
$ gcc main.c -o app.exe -L./lib -lBhavyTr -I./include
03
Run Your Application
$ ./app.exe
01
Open Terminal
Open Terminal.app and navigate to your project directory.
02
Run GCC / Clang Command
Compile with GCC or the system Clang compiler.
$ gcc main.c -o app -L./lib -lBhavyTr -I./include
03
Run Your Application
$ ./app
01
Open Terminal
Open your terminal in the folder containing your C file.
02
Run GCC Command
Compile and link the shared library.
$ gcc main.c -o app -L./lib -lBhavyTr -I./include -Wl,-rpath,./lib
03
Run Your Application
$ ./app

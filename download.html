Quick Start

unzip BhavyTr-v1.0.0.zip cd BhavyTr-v1.0.0/examples gcc add.c -I../include -L../lib -lbhavytr -o app ./app

Compile

Windows
Linux
macOS
gcc main.c -o app.exe -I./include -L./lib -lbhavytr ./app.exe
gcc main.c -o app -I./include -L./lib -lbhavytr -Wl,-rpath=./lib ./app
clang main.c -o app -I./include -L./lib -lbhavytr ./app

Requirements

  • GCC / Clang (C11)
  • Windows / Linux / macOS

Minimal Example

examples/add.c

#include <BhavyTr/core/Tensor.h> int main() { // Initialize tensor(s) // Perform operation // Output result return 0; }

Runtime Notes

  • Windows: keep libbhavytr.dll beside app.exe
  • Linux: use -Wl,-rpath=./lib
